# console-ui
## 简介
一个C++头文件，用于控制台用户界面程序的开发。
基于ANSI转义序列实现，支持Windows、Linux和macOS等平台。
> 此项目含AI生成内容。
## 仓库里有什么?
`test.cpp` - 测试程序源代码。

`console_ui.h` - 头文件本身。——为什么总感觉这个项目有点简陋？

`README.md` - 你现在在看的自述文件。——绝大多数GitHub仓库的标配！

`test` - 编译的测试程序(Linux环境)。

## 怎么用？
- 先从此仓库的Releases下载压缩包`console_ui_1.0.zip`。

- 再把`console_ui.h`放到你的项目目录里。

- 最后，在源代码文件里引用：
`#include "console_ui.h"`

- 接下来就可以使用了。

## 包含函数
`mvc(x,y)` - 将光标移动到指定位置（列 x, 行 y）。

`clrtxt(text,txtcolor,bgcolor)` - 以指定前景色和背景色输出文本，随后重置颜色。

`hc()` - 隐藏光标。

`sc()` - 显示光标。

`setup()` - 在 Windows 上启用虚拟终端（VT）处理以支持 ANSI 转义序列；其他平台（例如Linux和macOS）为 no-op。

`termh()` - 返回终端高度（行数），平台依赖，失败时返回 24（大部分终端默认高度）。

`termw()` - 返回终端宽度（列数），平台依赖，失败时返回 80（大部分终端默认宽度）。
